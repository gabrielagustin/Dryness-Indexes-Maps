# Dryness-Indexes-Maps
Generates dryness indexes maps using SMAP images. The indices applied are:

- Temperature Condition Index (TCI)
- Normalized Soil Moisture Index (HSCI)

Satellite Data:

- SMAP_L3_SM_P_E (9 km spatial resolution)
